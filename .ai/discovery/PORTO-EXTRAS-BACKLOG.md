# Discovery - Porto Extras Backlog

## Objetivo

Registrar extras da Porto que podem ser importantes depois, sem bloquear a primeira implementacao do core AUTO.

## Categorias

- Core now: entra na primeira implementacao Porto.
- Broker review: aparece para o corretor revisar, mas nao necessariamente para o segurado.
- Client output: pode aparecer no PDF/link publico.
- Future tier: recurso de valor para plano futuro ou tela avancada.
- Ignore for now: dado tecnico ou pouco util no momento.
